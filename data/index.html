ESP8266 Hakkında

ESP8266, düşük maliyetli, düşük güç tüketimli bir Wi-Fi mikrodenetleyici çipidir. Çeşitli dijital giriş/çıkış pinleri, bir analog giriş pini ve çok sayıda iletişim arabirimine sahiptir.

D1 (GPIO5)

Dijital I2C SCL

Dijital giriş/çıkış olarak kullanılabilir. I2C haberleşmesinde SCL (Clock) olarak da kullanılabilir.

  • Maksimum Akım: 12mA
  • Giriş Voltajı: 0-3.3V (toleranslı değil)
  • Pull-up/down: Dahili pull-up direnci

D2 (GPIO4)

Dijital I2C SDA

Dijital giriş/çıkış olarak kullanılabilir. I2C haberleşmesinde SDA (Data) olarak da kullanılabilir.

  • Maksimum Akım: 12mA
  • Giriş Voltajı: 0-3.3V (toleranslı değil)
  • Pull-up/down: Dahili pull-up direnci

D3 (GPIO0)

Dijital Özel

Dijital giriş/çıkış olarak kullanılabilir. Boot modu için kullanıldığından, bu pin bootup sırasında LOW olursa, ESP8266 flash moduna girer.

  • Maksimum Akım: 12mA
  • Giriş Voltajı: 0-3.3V (toleranslı değil)
  • Özel Fonksiyon: Flash modu tetikleyicisi

D4 (GPIO2)

Dijital LED_BUILTIN

Dijital giriş/çıkış olarak kullanılabilir. NodeMCU üzerindeki dahili LED bu pine bağlıdır (LOW aktiftir).

  • Maksimum Akım: 12mA
  • Giriş Voltajı: 0-3.3V (toleranslı değil)
  • Özel Fonksiyon: Boot sırasında HIGH olmalıdır

D5 (GPIO14)

Dijital SPI SCK

Dijital giriş/çıkış olarak kullanılabilir. SPI haberleşmesinde SCK (Clock) olarak da kullanılabilir.

  • Maksimum Akım: 12mA
  • Giriş Voltajı: 0-3.3V (toleranslı değil)

A0 (ADC0)

Analog

ESP8266'daki tek analog giriş pinidir. Potansiyometre, LDR, sıcaklık sensörü gibi analog sensörleri okumak için kullanılabilir.

  • Giriş Voltajı: 0-1V (NodeMCU'da dahili voltaj bölücü sayesinde 0-3.3V)
  • Çözünürlük: 10-bit (0-1023 değer aralığı)
  • Not: ESP8266 analog çıkış (DAC) özelliğine sahip değildir. Analog çıkış gerekiyorsa PWM kullanılabilir.
0V = 0 3.3V = 1023

RX (GPIO3)

UART

Seri haberleşme için alıcı (RX) pinidir. USB-Seri dönüştürücüye bağlıdır.

  • Not: Serial.print() komutları bu pini kullanır
  • Diğer Kullanımlar: Normal GPIO olarak da kullanılabilir

TX (GPIO1)

UART

Seri haberleşme için verici (TX) pinidir. USB-Seri dönüştürücüye bağlıdır.

  • Not: Serial.print() komutları bu pini kullanır
  • Diğer Kullanımlar: Normal GPIO olarak da kullanılabilir

Güç Pinleri

Güç

ESP8266 NodeMCU kartı üzerindeki güç pinleri:

  • 3V3: 3.3V güç çıkışı
  • GND: Toprak pini
  • Vin: Harici güç girişi (5V uygulanabilir)

ESP8266 I/O pinleri 5V toleranslı değildir! Daima 3.3V seviyesinde çalışın.

RST

Sistem

Reset pini. LOW yapıldığında ESP8266 yeniden başlatılır.

EN (CH_PD)

Sistem

Chip Enable pini. LOW yapıldığında ESP8266 devre dışı kalır.

İpucu: ESP8266 ile projelerde 3.3V lojik seviyelerine dikkat edin.

ESP8266 - Yardım ve Örnekler

ESP8266 ile Başlangıç

ESP8266, düşük maliyetli bir WiFi çözümü sunan popüler bir IoT geliştirme kartıdır. Bu dashboard ile ESP8266'nızın GPIO pinlerini kolayca kontrol edebilir ve izleyebilirsiniz.

Dashboard Kullanımı

  • Çıkış Pinleri: Bu kartlar, ESP8266'nın dijital pinlerini kontrol etmenizi sağlar. Anahtarları açıp kapatarak pinleri HIGH veya LOW durumuna getirebilirsiniz.
  • Giriş Pinleri: Bu kartlar, ESP8266'nın analog pininden okunan değerleri gösterir. Örneğin, potansiyometre, LDR veya diğer analog sensörlerden gelen verileri görebilirsiniz.
  • Yenileme: Dashboard 1 saniyede bir otomatik olarak yenilenir. Manuel yenilemek için sağ üstteki yenileme butonuna tıklayabilirsiniz.
  • Tema: Aydınlık ve karanlık tema arasında geçiş yapmak için güneş/ay simgesini kullanabilirsiniz.

Bağlantı Bilgileri

ESP8266 iki modda çalışabilir:

  • Access Point (AP) Modu: ESP8266 kendi WiFi ağını oluşturur. Varsayılan SSID: ESP_Dashboard, Şifre: 12345678
  • Station (STA) Modu: ESP8266 mevcut bir WiFi ağına bağlanır. Bunun için kaynak kodunda WiFi bilgilerini yapılandırmanız gerekir.

Örnek Projeler

LED Kontrolü

D1, D2, D3, D4 veya D5 pinlerine LED'ler bağlayın ve dashboard üzerinden kontrol edin. 


// LED'i D2 (GPIO4) pinine bağlama
const int ledPin = D2;

void setup() {
  pinMode(ledPin, OUTPUT);
}

// Dashboard üzerinden kontrol ediliyor

Bağlantı Şeması:

  • LED + (Anot) → 220Ω direnç → D2
  • LED - (Katot) → GND

Potansiyometre ile Analog Okuma

Potansiyometre bağlayarak analog değerler okuyun ve dashboard üzerinde görüntüleyin. 


// Potansiyometreyi A0 pinine bağlama
const int potPin = A0;

void setup() {
  // A0 pini otomatik olarak analog giriş olarak ayarlanır
}

// Dashboard otomatik olarak değerleri okuyor

Bağlantı Şeması:

  • Potansiyometre pin 1 → 3.3V
  • Potansiyometre pin 2 → A0
  • Potansiyometre pin 3 → GND

Röle Kontrolü

Röle modülü bağlayarak elektrikli cihazları kontrol edin.


// Röleyi D1 pinine bağlama
const int relayPin = D1;

void setup() {
  pinMode(relayPin, OUTPUT);
}

// Dashboard üzerinden kontrol ediliyor

Bağlantı Şeması:

  • Röle IN pini → D1
  • Röle VCC pini → 3.3V
  • Röle GND pini → GND

Sensör Bağlantıları

DHT11/DHT22 Sıcaklık ve Nem Sensörü

DHT sensörleri, sıcaklık ve nem ölçümü için yaygın olarak kullanılır. 


#include "DHT.h"
#define DHTPIN D4
#define DHTTYPE DHT11 // DHT11 veya DHT22

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  dht.begin();
}

void loop() {
  float h = dht.readHumidity();
  float t = dht.readTemperature();
  // Değerleri kullanın
}

Bağlantı: VCC → 3.3V, DATA → D4, GND → GND

LDR (Işık Sensörü)

LDR ile ortam ışığı ölçümü yapabilirsiniz.


// LDR'yi A0 pinine bağlama
// 10K direnç ile voltaj bölücü devre kullanılıyor

void setup() {
  // A0 pini otomatik olarak analog giriş olarak ayarlanır
}

void loop() {
  int lightValue = analogRead(A0);
  // Değeri kullanın
}

Bağlantı: LDR bir ucu → 3.3V, LDR diğer ucu → A0 ve 10K direnç → A0 ve GND

HC-SR04 Ultrasonik Mesafe Sensörü

Ultrasonik sensör ile mesafe ölçümü yapabilirsiniz.


const int trigPin = D1;
const int echoPin = D2;

void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
}

long readDistance() {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  
  long duration = pulseIn(echoPin, HIGH);
  return duration * 0.034 / 2; // cm cinsinden mesafe
}

Bağlantı: VCC → 3.3V, TRIG → D1, ECHO → D2, GND → GND

Not: Bu sensör 5V ile daha iyi çalışır, ancak sinyal pinleri 3.3V ile uyumludur.

Sorun Giderme

WiFi Bağlantı Sorunları

  • ESP8266 AP modunda bağlanılamıyor: ESP8266'yı yeniden başlatın, güç kaynağının yeterli olduğundan emin olun.
  • STA modunda internet bağlantısı yok: WiFi kimlik bilgilerini kontrol edin, modem/router'ın çalıştığından emin olun.
  • Bağlantı kesiliyor: ESP8266 ve router arasındaki mesafeyi kontrol edin, güç kaynağını kontrol edin.

GPIO Pin Sorunları

  • Pin kontrolü çalışmıyor: Bağlantıları kontrol edin, doğru pin numarasını kullandığınızdan emin olun.
  • Analog okumalar hatalı: Voltaj bölücü devreyi kontrol edin, referans voltajının (3.3V) stabil olduğundan emin olun.
  • Pinler hasar görebilir: GPIO pinlerine 3.3V üzerinde voltaj uygulamayın.

Dashboard Sorunları

  • Dashboard yüklenmiyor: Doğru IP adresinden eriştiğinizden emin olun, tarayıcı önbelleğini temizleyin.
  • Kontroller çalışmıyor: JavaScript hatası olabilir, tarayıcı konsolunu kontrol edin.
  • Data güncellenmiyor: Bağlantı problemleri olabilir, ESP8266'yı yeniden başlatın.
